# Commit Messages

## Tipos comuns

- `feat` → nova feature
- `fix` → correção de bug
- `chore` → tarefas de manutenção / dependências
- `docs` → documentação
- `refactor` → refatoração de código

## Exemplos

- `feat(auth): adiciona cadastro de usuário`
- `fix(ci): corrige instalação de dependências no workflow`
- `chore: atualizar dependências`
